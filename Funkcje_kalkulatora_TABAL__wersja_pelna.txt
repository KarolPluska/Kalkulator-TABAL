- Funkcje aktywne - konfigurator i wejście:
  - Pole liczby: Długość ogrodzenia (m) z własnym spinnerem góra/dół.
  - Lista: System ogrodzenia (System Fokus, System Fokus Light, System Flat, System Aluminium Prime).
  - Lista: Model - dynamiczna, zależna od wybranego systemu.
  - Lista: Wysokość przęsła (mm) - dynamiczna, zależna od systemu/modelu.
  - Lista: Kolor ogrodzenia (RAL) - dynamiczna, zależna od systemu.
  - Wariant koloru: Inny (Na zamówienie) z obsługą wyceny indywidualnej.
  - Lista: Słupek systemowy / opcja systemowa (różna logika dla paneli i Aluminium).
  - Dla opcji "Własny rozmiar (murowane/betonowe)": przycisk "+" przy selekcie słupka.
  - Modal "Własny rozmiar słupka":
    - Pole liczby: szerokość słupka (mm).
    - Walidacja minimalnej wartości.
    - Zapis/Anuluj/Zamknij.
    - Aktualizacja tytułu/przycisku z bieżącą wartością mm.

- Funkcje aktywne - pola dynamiczne dla Aluminium Prime:
  - Sekcja bram/furtek widoczna tylko dla systemu Aluminium Prime.
  - Lista: Furtka (wariant) + Ilość furtki.
  - Lista: Brama dwuskrzydłowa (wariant) + Ilość bram dwuskrzydłowych.
  - Lista: Brama przesuwna (wariant) + Ilość bram przesuwnych.
  - Pola ilości bram/furtek:
    - Domyślnie zablokowane.
    - Aktywują się dopiero po wyborze wariantu.
    - Mają własne etykiety i spinner góra/dół.

- Funkcje aktywne - wycena:
  - Wyliczanie ceny jednostkowej i sumy dla każdej pozycji.
  - Obsługa cen nieznanych:
    - "brak ceny".
    - "Wycena indywidualna".

    - Nie zawyżają sumy końcowej.

    - Dodawana jako osobny produkt w podsumowaniu.

  - Tabela z kolumnami:
    - Element.
    - Ilość.
    - Cena jedn.
    - Suma.
    - Rabat %.
    - Po rabacie.
    - Funkcje.
  - Każdy wiersz produktu:
    - Edycja ilości przez spinner góra/dół.
    - Akcje w kolumnie "Funkcje":
      - Check (zapis/odznacz zapisanie pozycji).

    - Wizualnie wydzielona separatorem i nagłówkiem.

      - Check (status zapisania).
      - X (usuń zapisaną pozycję).
      - Powiel (duplikacja zapisanej pozycji).
  - Zasada check dla bieżącej konfiguracji:
    - Zielony check tylko gdy istnieje dokładnie taka sama zapisana pozycja.
    - Zmiana ilości/parametrów powoduje utratę zgodności.
    - Ponowne kliknięcie check może dodać nową pozycję zapisaną (gdy konfiguracja się różni).
